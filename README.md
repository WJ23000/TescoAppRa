# TescoAppRa

## Project create
```
npx react-native init TescoAppRa
```
## Project setup
```
npm install
```

### Compiles and hot-reloads for android
```
react-native run-android
```

### Compiles and hot-reloads for android
```
react-native run-ios
```

### 页面路径
```
首页：HomePage
分类页：ClassPage
购物车：CartPage
个人中心：MinePage
```
